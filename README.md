# OpenLibrary Book-Filter

این پروژه یک اسکریپت پایتون است که ۵۰ کتاب را از API سایت OpenLibrary دریافت کرده و کتاب‌هایی که سال انتشار آن‌ها بعد از ۲۰۰۰ است را در یک فایل CSV ذخیره می‌کند.

## نحوه اجرا
۱. ابتدا کتابخانه‌های مورد نیاز را نصب کنید:
`pip install -r requirements.txt`

۲. اسکریپت را اجرا کنید:
`python main.py`

## خروجی
خروجی در فایل `books.csv` ذخیره خواهد شد.
